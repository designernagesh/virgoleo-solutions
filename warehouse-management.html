Warehouse Management System (WMS)

    Functions & Key benefits
  • Master data การตั้งค่าระบบ
  • Standard rate profile/Quotation/Vendor rate มีระบบใบเสนอราคา ซึ่งเชื่อมต่อกับข้อมูลลูกค้า ข้อมูลสินค้า และการกำหนดเงื่อนไขราคาในระบบ เช่น billing rule, credit term, credit limit
  • Multi-Warehouse รองรับการทำงานหลายคลังพร้อมกันและสามารถควบคุมการจัดเก็บสินค้าโดยการกำหนดเงื่อนไขการจัดเก็บสินค้าภายในคลังสินค้าได้หลากหลาย เช่น Warehouse location/zone,location sequence, storage type, storage unit
  • รองรับกระบวนการส่งผ่านสินค้าหรือ Cross Docking
  • Order entryการจอง warehouse
  • Tally In การรับสินค้าเข้าคลัง
  • Put Away การจัดเก็บสินค้า ระบบแนะนำตำแหน่งการจัดเก็บในคลังสินค้าที่เหมาะสมที่สุดให้
  • Shipping Instruction (SI) การจองสินค้าที่จะนำออกจากคลัง
  • Picking การนำสินค้าออกจากที่เก็บ ควบคุมการจ่ายสินค้าแบบ FIFO, LIFO
  • Tally Out การนำสินค้าออกจากคลัง
  • Load manifest การขนส่งสินค้า สามารถกำหนดเงื่อนไขการจัดการขนส่งเพื่อให้ระบบช่วยวางแผนการขนส่งให้
  • Work order การโอนย้ายสินค้า เช่น location transfer, stock room transfer
  • Stock adjustment การปรับยอดสินค้าคงเหลือ ในกรณีที่มีสินค้าทดแทน (stock replenishment),การรวมพาเลต (merge pallet), การแยกพาเลต (split pallet), kitting
  • Cycle count การตรวจนับสินค้าตามที่กำหนดรอบการนับของสินค้านั้นๆ
  • Cargo detail correction การแก้ไขรายละเอียดสินค้า
  • กำหนด Stock replenishment, Stock reorder ได้
  • Customer invoice การออกใบแจ้งหนี้
  • Storage invoice monitoring การติดตามลูกหนี้
  • Auto invoice ระบบการแจ้งหนี้อัตโนมัติ ตามเงื่อนไขที่กำหนด และแสดงสถานะเตือนให้ user ทราบ
  • Auto report ระบบการส่งรายงานอัตโนมัติ
  • EDI/Data interface สามารถ interface กับ third party system เช่น accounting system
  • รองรับระบบ barcode, RFID
