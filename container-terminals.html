Container Terminal System (ICD & Depot)

Inland Container Depot & Container Freight Station Management System caters to specific needs of CFS and ICD operators. Providing all the functionality of booking pre-advice, multiple quotation, gate operation, CFS operation, invoicing, cash collection, yard management, Rail/truck operations, Inventory management, truck monitoring, container inquiry plus EDI communication, Web inquiry, real-time operational /management report and Data interface with legacy system. The application enables the ICD and Depot operator to manage the cash collection and issue the credit invoice based on container movements and actual activities. Control truck-turnaround time and minimize human-error.

    Key Benefits
  • สร้าง standard rate profile และ quotation ในระบบได้ครอบคลุมทุกเงื่อนไขเพื่อรองรับการเรียกเก็บค่าบริการทุกรูปแบบ ตามข้อตกลงกับ agent/freight forwarder/customerเพื่อความถูกต้องแม่นยำในการคำนวณค่าใช้จ่ายที่เรียกเก็บจากลูกค้า
  • สร้าง booking ได้สะดวกรวดเร็วด้วยระบบ EDI, Excel upload, pdf file, ไฟล์ภาพทุกรูปแบบ หรือ manual
  • สร้างตารางการเดินเรือ vessel schedule ในระบบ เพื่อกำหนดและบริหารจัดการตู้เข้า-ออก ตาม cut off time
  • กำหนด status ตู้, hold ตู้, fix port, จองตู้ล่วงหน้า สามารถคัดแยกประเภทตู้ ด้วยการกำหนด status ตู้ได้
  • Pre Gate in เพื่ออัพเดท container status/damage condition ตามการเซอร์เวย์เบื้องต้นก่อนการรับตู้เข้าลาน ด้วยระบบกราฟฟิก (android application) โดยข้อมูลเบื้องต้นจะลิงค์ไปที่ gate in
  • มีหน้าจอ Truck monitoring สำหรับใช้วัด Kpi ในการปล่อยตู้ช่วยให้สามารถเข้าไปแก้ไขปัญหาได้ทันทีที่เห็นทะเบียนรถที่เข้ามารอในลานนานเกินกว่าที่กำหนดใน kpi โดยระบบจะโชว์ทะเบียนรถทุกคัน, ข้อมูลเบื้องต้น และเวลาที่รอ ของรถที่เข้ามารอรับบริการแล้วยังคงรออยู่ในลาน
  • มี work order สำหรับการอัพเดท Precool, PTI ,ค่าไฟตู้หนัก สำหรับค่าไฟตู้หนักถ้าไม่ต้องการอัพเดท work order ระบบสามารถคำนวณอัตโนมัติได้
  • ระบบสามารถควบคุมการเรียกเก็บเงินจากลูกค้าอย่างถูกต้องครบถ้วนทุกตู้และทุกกิจกรรม ทั้งเครดิตและเงินสด โดยการคิดค่าใช้จ่ายอัตโนมัติทุกรายการตามที่กำหนดใน standard rate profile หรือ quotation เช่น ค่ายกตู้ ค่าผ่านทาง ค่าไฟตู้หนัก ค่า storage ,precool, PTI หรือค่าใช้จ่ายอื่นๆ ที่มีการเรียกเก็บ โดยสามารถกำหนด free time ได้ทั้ง เป็นวันและ TEU
  • สามารถออก Full TAX Invoice, cash bill, coupon, EIR , Pick up form อัตโนมัติหน้าเกทได้ทันที
  • รองรับการจ่ายเงินล่วงหน้า โดยระบบสามารถคำนวณค่า storage ล่วงหน้าได้ตามที่ระบุ
  • ระบบควบคุมการเก็บเงินสดในแต่ละวัน ของแต่ละ user name โดยสามารถเรียกดูยอดเงิน และรายละเอียดการเก็บเงินของแต่ละ user แบบ real time ได้ หรือจะเรียกข้อมูลตาม agent ก็ได้ เพื่อความโปร่งใสและความถูกต้องในการทำงานของพนักงาน
  • สามารถพัฒนาไปสู่การทำระบบ Auto gate โดยระบบสามารถอัพเดท gate in / out และออก cuopon/tax invoice อัตโนมัติ ได้โดยไม่ต้องใช้ user
  • auto send report การส่งรายงานอัตโนมัติโดยระบบ ช่วยลดเวลาและความผิดพลาดในการส่งรายงานให้ผู้ที่เกีายวข้อง เช่น สายเรือ ลูกค้า โดยสามรถตั้งเวลาการส่ง, ช่วงข้อมูลที่ต้องการส่ง, ชื่อรายงานที่ต้องการส่ง, email ของผู้รับ ระบบจะส่งรายงานให้โดยอัตโนมัติ ตามเงื่อนไขที่ตั้งไว้
  • EDI มีความถูกต้องแม่นยำ 100% ทั้ง Codeco ส่ง movement ตู้, Coparn รับข้อมูล booking หรือ excel upload
  • สามารถ generate barcode document สำหรับเอกสารต่างๆ ได้ เช่น coupon, tax invoice, eir เพื่อความถูกต้องและสะดวกรวดเร็วในการค้นหาข้อมูลในระบบโดยการ scan barcode แทนการ manual key in ซึ่งอาจมีการคีย์ข้อมูลผิดพลาดได้
