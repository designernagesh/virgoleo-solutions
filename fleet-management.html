Fleet Management System

Functions & Key Benefits
  • Truck Preventive Maintenance สามารถสร้างตารางการบำรุงรักษารถแต่ละคันได้ เพื่อให้ระบบเตือนล่วงหน้าก่อนถึงกำหนดการซ่อม
    • Equipment Profile กำหนดประวัติและข้อมูลของรถแต่ละคัน
    • Preventive Maintenance Scheduler สร้างตารางกำหนดการซ่อมบำรุง
    • Work Order สร้างใบสั่งซ่อม
    • Work Order Monitoring ติดตามการซ่อมบำรุง
  • Truck Monitoring การควบคุมและติดตามรถ
    • Mileage Usage ระยะทางการวิ่งของรถ
    • Traffic Violations การฝ่าฝืนกฏจราจร
  • Inventory / Spare Part การควมคุม stock การซื้อ รับ เบิก จ่าย อะไหล่ต่างๆ
    • Parts Inventory อะไหล่คงเหลือ
    • purchase requisition การเสนอขอซื้ออะไหล่
    • Receive Goods การรับอะไหล่เข้าคลัง
    • Purchase Order การออกใบสั่งซื้อ
  • Costing / Billing ค่าใช้จ่ายและการออก invoice
    • Work Order Invoice ออก invoice สำหรับการซ่อมแต่ละครั้ง
    • Vendor Invoice / Cost Entry การบันทึกค่าใช้จ่ายและทำจ่ายให้ซัพพลายเออร์
  • Inquiries ระบบการค้นหาข้อมูล
    • Fuel Usage Analysis วิเคราะห์อัตราการสิ้นเปลืองน้ำมันของรถแต่ละคัน
    • Tires Usage Analysis วิเคราะห์การใช้ยางของรถ
    • Equipment Mileage Analysis วิเคราะห์ระยะการวิ่งของรถ
    • Truck Through Put Analysis วิเคราะห์ kpi ประสิทธิภาพการทำงานของรถแต่ละคัน
    • Ad-hoc Repair Thread Analysis by Equipment วิเคราะห์การซ่อมบำรุงรถแต่ละคัน
    • Part Usage Analysis วิเคราะห์การใช้อะไหล่
  • Report รายงานต่างๆ เช่น รายงานประวัติการซ่อมรถแต่ละคัน ประวัติการเปลี่ยนยางของรถ ค่าซ่อมรายเดือน/รายปี อายุการใช้งานของยางใช้ได้กี่วันกี่กิโลเมตร การใช้อะไหล่ เป็นต้น
